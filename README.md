Lista 05 de Exercícios - Estrutura de Repetição For

Esta lista contém 15 exercícios de lógica de programação focados no uso da estrutura de repetição for em JavaScript. Os exercícios envolvem:

Contagem e exibição de números (pares, ímpares, de 1 a N)

Tabuada de um número fornecido pelo usuário

Soma de números e soma de múltiplos de 3

Cálculo de média, maior e menor valor de uma sequência

Sequência de Fibonacci

Cálculo de fatorial

Manipulação de vetores e exibição invertida

Desenho de figuras simples com asteriscos

O objetivo é praticar o uso de loops for, desenvolvendo raciocínio lógico, controle de repetição com contadores e manipulação de dados de entrada.
